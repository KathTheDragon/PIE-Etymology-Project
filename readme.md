# PIE Etymology Project
Essentially an etymological dictionary for the entire Indo-European family.

### How to use
Each folder contains a markdown file and a folder for static resources. The markdown file is the main dictionary document, sorted alphabetically by headword. Headwords should have an HTML element with an id to allow links to target specific headwords - the suggested id format is `"{headword}"` or `"{headword}-{homonym-number}"`. Each entry should be accompanied by a list of citations which should link to the corresponding entry in `bibliography.md` in the main folder. The body of the entry can be text or an image, but either way should contain links to any other referenced entry.
